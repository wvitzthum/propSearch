# UI/UX Scoping & Audit Report - March 7, 2026

## 1. Map Clarity Audit (QA-016) - [RESOLVED]
**Assessment:** The current `.retro-map` CSS implementation (found in `index.css`) applies global filters that compromise usability for expert research.
- **Findings:** Global `brightness(0.85)` and `grayscale(0.2)` on `.leaflet-container` dim and desaturate interactive markers, nodes, and spatial context overlays.
- **Recommendation:** 
  1. Remove `filter` from `.leaflet-container`.
  2. Apply `filter: brightness(0.9) contrast(1.1) grayscale(0.1)` ONLY to the `.leaflet-tile-pane` or the `TileLayer` component.
  3. Ensure markers and popups remain at 100% opacity and full saturation to preserve "High Signal" data visibility.

## 2. Market Pulse Scoping (QA-018) - [RESOLVED]
**Aesthetic Goal:** Bloomberg Terminal (High density, KPI-heavy, minimal sparklines).
- **Data Source:** `data/macro_trend.json`.
- **Layout Proposal:**
  - **KPI Header (4 Nodes):** HPI % (MoM/YoY), Months of Supply (Velocity), Negotiation Delta (Discount %), BoE Base Rate.
  - **Sparklines:** Small trend lines showing Listing vs Sold volume from `business_history` (12-24mo).
  - **Heat Index List:** High-contrast list of areas with 0-10 scores and trend arrows (Rising/Stable/Falling).
  - **Analyst Note:** Text block for `market_pulse_summary`.
- **Placement:** Top of Dashboard (collapsible) or a new widget in the Sidebar.

## 3. Landing Page "Market Situation Room" (QA-020) - [RESOLVED]
**Aesthetic Goal:** Linear-grade precision + Bloomberg data density.
- **Hero Section:** High-level market status (e.g., "Buyer-led / Optimal Buy Window").

## 4. Critical Data Desync (QA-019) - [RE-SCOPED]
- **Bug:** `data/master.json` provides an array `links: string[]`, but `frontend/src/types/property.ts` and UI components expect `link: string`.
- **Impact:** All "Go to Source" links and Detail page buttons are currently broken/non-functional.
- **Action for Data Agent:** Update `generate_data.py` or `master.json` to provide a singular `link` property (preferably the primary Rightmove/Zoopla URL).
- **Action for Frontend Agent:** Update `Property` interface and component props to handle either a singular link or mapping the first item of the array.
