### Root Cause Analysis

**UI-003 (Critical):** The `macro_trend.json` schema was updated with provenance metadata wrapping values in objects:
```json
// OLD (expected by hook):
"boe_base_rate": 3.75
"mortgage_history": [...]

// NEW (actual in file):
"boe_base_rate": { "value": 3.75, "source": "..." }
"mortgage_history": { "data": [...] }
```

The `useFinancialData.ts` hook's `normalizeMacro()` function doesn't handle the new schema structure.
