# HM Land Registry Source Citation

## Data Used
- **UK House Price Index (HPI):** Mix-adjusted average house prices
- **Annual/Monthly Change:** Percentage changes in property values
- **Regional Data:** Area-specific price trends for London postcodes

## Source URLs
- **Main:** https://landregistry.data.gov.uk/
- **HPI Data:** https://landregistry.data.gov.uk/app/hpi
- **Price Paid Data:** https://landregistry.data.gov.uk/app/ppd

## Methodology Notes
- HPI uses mix-adjustment to account for changes in property mix
- Data typically has 2-3 month lag for finalization
- Monthly indices are provisional until revised
